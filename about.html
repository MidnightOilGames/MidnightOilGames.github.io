About the Game

A cozy art dealership simulator built by Midnight Oil.

What is Going Once!?

Going Once! is a narrative-light management sim where you run an art auction house: inspecting artwork, building relationships with buyers, and running live high-stakes auctions where every choice affects your reputation.

INSPECTING ART

Use professional tools to uncover authenticity.

Rotate, zoom, and examine each piece in detail using tools. Find mismatched signatures, hidden markings, overpainting, and material inconsistencies.
